Description

Features are the essential building blocks of data analysis and machine learning insights. Numerical and categorical features are two basic features that play a critical role in this process.

Numerical and categorical features have different characteristics and require other analysis methods. The preprocessing and transformations you perform will depend on the type of features. Your chosen machine-learning algorithm will also depend on the target variable type.

In this project, you will build and optimize predictive models with the Medical Insurance dataset from Kaggle. All you have to do in this stage is to tell apart the numerical features from the categorical ones.

Objectives

To complete this stage:

  1. Call the provided download_data() function to download the medical insurance dataset and save it as insurance.csv in the data directory;
  2. Load the data in a pandas DataFrame;
  3. Remove duplicates from the data with the .drop_duplicates() method;
  4. Print a dictionary containing a list of the numerical and categorical features and the shape of the data.

As an optional activity, you can make univariate and bivariate plots for the numerical and categorical features to understand your dataset. Tests will not check this.

When you call the download_data() function, the data directory is created and the insurance.csv file is downloaded and saved in it.
